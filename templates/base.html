django-treenav

To view a demo, create a new menu tree in the Django admin and set the root MenuItem slug to "primary-nav", and its link to "/". Create as many child MenuItems under the root making sure that the link values for each child is a valid child link of its parent. For example, you could create a root MenuItem with a link of "/", a child of the root with a link of "/one", and a child of that with a link of "/one/two". Django Treenav uses the values in the links to determine where in the hierarchy you are.
